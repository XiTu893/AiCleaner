# 智清大师 - 管理员权限运行说明

## ⚠️ 为什么需要管理员权限？

智清大师的以下功能需要管理员权限才能正常工作：

1. **温度监控** - 读取硬件传感器数据需要访问 WMI
2. **软件卸载** - 读取注册表和卸载程序需要管理员权限
3. **系统清理** - 清理系统文件需要更高权限

## 🚀 如何以管理员身份运行

### 方法一：使用 PowerShell 脚本（推荐）

在项目目录下运行：

```powershell
.\run-as-admin.ps1
```

### 方法二：手动提升权限

1. 找到项目目录：`E:\zzp\project\github\AI\AICLeaner`
2. 右键点击 `run-as-admin.ps1`
3. 选择"使用 PowerShell 运行"
4. 在 UAC 提示中点击"是"

### 方法三：在 VSCode/Trae 中以管理员身份运行

1. 关闭当前的编辑器
2. 右键点击编辑器图标
3. 选择"以管理员身份运行"
4. 重新打开项目并运行 `bun run dev`

## 📋 功能对比

### 普通用户模式
- ✅ 硬件检测（CPU、内存、磁盘）
- ✅ 性能跑分
- ⚠️ 显卡检测（部分信息）
- ❌ 温度监控（拒绝访问）
- ❌ 软件卸载（权限不足）
- ❌ 系统清理（权限不足）

### 管理员模式
- ✅ 硬件检测（完整信息）
- ✅ 性能跑分
- ✅ 显卡检测（完整信息）
- ✅ 温度监控（CPU 温度）
- ✅ 软件卸载（完整功能）
- ✅ 系统清理（完整功能）

## 🔧 故障排除

### 问题：软件列表为空
**解决方案**：以管理员身份运行应用

### 问题：温度显示"拒绝访问"
**解决方案**：以管理员身份运行应用

### 问题：显卡信息为空
**解决方案**：
1. 以管理员身份运行应用
2. 检查 WMIC 是否可用：运行 `wmic path win32_VideoController get name`

### 问题：卸载软件失败
**解决方案**：
1. 以管理员身份运行应用后重试
2. 或手动在控制面板中卸载
3. 某些软件不支持静默卸载，需要手动操作
