# 智清大师 (AI Cleaner Pro) - 功能规划文档

## 📋 项目概述

**智清大师**是一款集硬件检测、性能评测、系统优化、驱动管理、硬件防护于一体的综合电脑管理工具。

### 技术栈
- **框架**: Electron 28 + React 18 + TypeScript
- **运行时**: Bun 1.3.10
- **构建工具**: Vite 5
- **UI 库**: Lucide React (图标)
- **系统 API**: Node.js 原生模块 + Windows API

### 窗口配置
- **默认尺寸**: 960 × 650 像素
- **最小尺寸**: 800 × 500 像素
- **菜单栏**: 自动隐藏

### 五、软件管理

#### 5.1 软件卸载 (Software Uninstaller)
**功能实现**
- ✅ 读取注册表获取软件列表
- ✅ 支持 HKLM 和 HKCU 两个注册表路径
- ✅ 显示软件详细信息
  - 软件名称（支持中文，无乱码）
  - 厂商信息
  - 版本号
  - 安装日期
  - 卸载命令
- ✅ 搜索功能
- ✅ 一键卸载（调用软件自带卸载程序）
- ✅ 支持 MSI 和 EXE 卸载程序
- ✅ 静默卸载参数优化
- ✅ 错误处理友好

**技术实现**
- ✅ 使用 `reg query` 命令读取注册表
- ✅ 支持 GBK 和 UTF-8 编码转换
- ✅ 使用 `iconv-lite` 处理中文乱码
- ✅ 详细的日志输出
