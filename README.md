# 智清大师 (AI Cleaner Pro)

一款基于 Electron + React 的系统工具软件，集硬件检测、性能评测、系统优化、驱动管理、硬件防护于一体的综合电脑管理工具。

## ✨ 核心功能

### 🔍 一、硬件检测

#### 硬件体检

- 一键扫描整机配置
- 生成硬件健康报告
- 提示潜在硬件问题

#### 硬件参数

- 精准识别所有硬件信息：
  - CPU：型号、核心数、频率、缓存
  - 显卡：GPU 型号、显存容量、频率
  - 主板：型号、芯片组、BIOS 版本
  - 内存：容量、频率、时序、品牌
  - 硬盘：型号、容量、接口类型、健康度
  - 显示器：尺寸、分辨率、面板类型
  - 网卡/声卡：型号、驱动版本
- 实时硬件状态监控

#### 真伪鉴别

- 对比官方参数数据库
- 识别显卡虚标、翻新
- 检测 CPU 是否正品
- 验证内存/硬盘真伪

#### 屏幕检测

- 坏点检测
- 漏光测试
- 对比度测试
- 色彩还原度测试

### 🏆 二、性能评测与跑分

#### 综合跑分

- CPU 基准测试
- 显卡 3D 性能测试
- 内存带宽测试
- 硬盘读写速度测试
- 综合性能评分

#### 分项评测

- CPU 运算能力测试
- 显卡 3D 渲染性能
- 内存读写带宽
- 硬盘顺序/随机读写速度

#### 性能排行

- 与全球同配置机型对比
- 性能水平直观评估
- 历史跑分记录对比

### 🛡️ 三、硬件监控与防护

#### 实时温度监控

- CPU 温度实时显示
- 显卡温度监控
- 主板温度监控
- 硬盘温度监控
- 高温自动预警

#### 压力测试

- CPU 满载烤机测试
- 显卡压力测试
- 系统稳定性检验
- 散热能力评估

#### 硬件健康

- 硬盘使用时间统计
- 硬盘坏道检测
- SSD 健康度评估
- 笔记本电池健康度

#### 节能降温

- 智能功耗调节
- 风扇转速控制
- 降温节能建议
- 延长硬件寿命

### ⚡ 四、系统优化与清理

#### 垃圾清理

- 系统临时文件清理
- 应用缓存清理
- 注册表冗余清理
- 大文件扫描与清理

#### 优化加速

- 开机启动项管理
- 系统服务优化
- 一键内存释放
- 游戏模式加速

#### C 盘瘦身/搬家

- 系统盘空间分析
- 大文件转移
- 软件迁移
- 释放系统盘空间

#### 桌面整理

- 保持桌面整洁

#### 内存释放

- 智能内存整理
- 后台进程优化
- 提升运行流畅度
