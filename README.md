## ✨ 核心功能

### 🔍 一、硬件检测

#### 硬件体检

- 一键扫描整机配置
- 生成硬件健康报告
- 提示潜在硬件问题

#### 硬件参数

- 精准识别所有硬件信息：
  - CPU：型号、核心数、频率、缓存
  - 显卡：GPU 型号、显存容量、频率
  - 主板：型号、芯片组、BIOS 版本
  - 内存：容量、频率、时序、品牌
  - 硬盘：型号、容量、接口类型、健康度
  - 显示器：尺寸、分辨率、面板类型
  - 网卡/声卡：型号、驱动版本
- 实时硬件状态监控

#### 屏幕检测

- 坏点检测
- 漏光测试
- 对比度测试
- 色彩还原度测试

### 🏆 二、性能评测与跑分

#### 综合跑分

- CPU 基准测试
- 显卡 3D 性能测试
- 内存带宽测试
- 硬盘读写速度测试
- 综合性能评分

#### 分项评测

- CPU 运算能力测试
- 显卡 3D 渲染性能
- 内存读写带宽
- 硬盘顺序/随机读写速度

#### 性能排行

- 与全球同配置机型对比
- 性能水平直观评估
- 历史跑分记录对比

### 🛡️ 三、硬件监控与防护

#### 实时温度监控

- CPU 温度实时显示
- 显卡温度监控
- 主板温度监控
- 硬盘温度监控
- 高温自动预警

#### 压力测试

- CPU 满载烤机测试
- 显卡压力测试
- 系统稳定性检验
- 散热能力评估

#### 硬件健康

- 硬盘使用时间统计
- 硬盘坏道检测
- SSD 健康度评估
- 笔记本电池健康度

#### 节能降温

- 智能功耗调节
- 风扇转速控制
- 降温节能建议
- 延长硬件寿命

### ⚡ 四、系统优化与清理

#### 垃圾清理

- 系统临时文件清理
- 应用缓存清理
- 注册表冗余清理
- 大文件扫描与清理

#### 优化加速

- 开机启动项管理（支持按名称、类型、状态排序）
- 系统服务优化（支持按名称、类型、状态排序）
- 一键内存释放
- 游戏模式加速
- 进程列表管理（支持按名称、内存排序，显示进程路径）

#### C 盘瘦身/搬家

- 系统盘空间分析
- 大文件转移
- 软件迁移
- 释放系统盘空间

#### 桌面整理

- 保持桌面整洁

#### 内存释放

- 智能内存整理
- 后台进程优化
- 提升运行流畅度

### 🔧 五、软件与驱动管理

#### 驱动备份/还原

- 一键备份当前驱动
- 旧版驱动恢复

#### 强力卸载

- 彻底删除软件
- 清理注册表残留
- 删除程序残留文件
- 卸载痕迹清理

### 🧰 六、实用工具集

#### 功耗计算器

- 估算整机功耗
- 指导电源选择
- 硬件功耗对比
- 电源推荐

#### 桌面监控

- 悬浮窗显示硬件数据
- 实时 CPU/内存占用
- 温度实时显示
- 网速监控

#### 数据恢复

- 恢复误删文件，照片恢复，文档恢复，支持多种文件格式

#### 文件粉碎机

- 彻底删除敏感文件，多次覆盖写入，无法恢复，支持 DoD 5220.22-M 标准

## 🛠️ 技术栈

- **框架**: Electron 28 + React 18 + TypeScript
- **运行时**: Bun 1.3.10
- **构建工具**: Vite 5
- **UI 库**: Lucide React (图标)
- **系统 API**: Node.js 原生模块 + Windows API
